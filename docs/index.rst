Welcome to okscraper-django's documentation!
============================================

Okscraper-django is a python module that provide django integration for okscraper.

Okscraper is a scraping framework and okscraper-django adds some features such as logging scraper runs to the database, displaying logs in the django admin interface, running scrapers via management commands.

If you are new to the okscraper framework you should check out the `okscraper documentation <http://okscraper.readthedocs.org/>`_ first.
